居中布局

以下居中布局均以不定宽为前提，定宽情况包含其中

水平居中

inline-block + text-align

table + margin

absolute + transform

如果子元素为定宽元素，则可以如下写法

flex + justify-content

垂直居中

table-cell + vertical-align

absolute + transform

flex + align-items

水平垂直居中

inline-block + table-cell + text-align + vertical-align

absolute + transform

flex
